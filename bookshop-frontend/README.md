# Bookshop Frontend - Pahana Bookshop

React-based frontend application for the bookshop e-commerce platform.

## 📋 Requirements

-   **Node.js**: 18.0 or higher
-   **npm or yarn**: Latest version
-   **Backend API**: Bookshop backend running on port 8080

## 🛠️ Setup

1. **Clone the repository**

    ```bash
    git clone <repository-url>
    cd bookshop-frontend
    ```

2. **Install dependencies**

    ```bash
    npm install
    # or
    yarn install
    ```

3. **Configure API endpoint**
    - The API is configured to connect to `http://localhost:8080/api`
    - Make sure the backend is running on port 8080

## 🚀 How to Run

1. **Start development server**

    ```bash
    npm run dev
    # or
    yarn dev
    ```

2. **Application will be available at**

    ```
    http://localhost:5173
    ```

3. **Build for production**
    ```bash
    npm run build
    # or
    yarn build
    ```

## 🔑 Default Login Credentials

-   **Admin Account:**
    -   Email: `admin@bookshop.com`

## 📱 Features

-   Browse and search books
-   Add books to cart
-   User registration and login
-   Place orders and checkout
-   Admin panel for managing books, users, and orders
-   Responsive design for mobile and desktop
